Empieza con Auth Spring Boot Starter en menos de 5 minutos.

## Prerrequisitos

- Java 17 o superior
- Spring Boot 3.x
- Una base de datos (PostgreSQL, MySQL, etc.)

## Instalación

Añade la dependencia a tu proyecto.

::code-group
  ```xml [pom.xml]
  <dependency>
      <groupId>dev.ctlabs</groupId>
      <artifactId>auth-spring-boot-starter</artifactId>
      <version>0.0.1-SNAPSHOT</version>
  </dependency>
  ```

  ```groovy [build.gradle]
  implementation 'dev.ctlabs:auth-spring-boot-starter:0.0.1-SNAPSHOT'
  ```
::

## Configuración Básica

Añade la siguiente configuración a tu `application.yml` o `application.properties`:

```yaml
ctlabs:
  auth:
    jwt:
      secret-key: ${JWT_SECRET:tu-secreto-de-256-bits} # Debe ser una cadena Base64 larga y segura
      expiration: 86400000 # 24 horas en milisegundos
    frontend-url: ${FRONTEND_URL:http://localhost:3000}
```

## Ejecuta Tu Aplicación

El starter autoconfigura todos los componentes necesarios y crea el esquema de base de datos automáticamente usando Flyway.

Ahora puedes acceder a la documentación de la API en `/api/auth` (o tu ruta base configurada).
