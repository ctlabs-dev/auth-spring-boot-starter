El starter expone los siguientes endpoints bajo la ruta base configurada (por defecto: `/api/auth`).

| Método | Endpoint | Descripción | Auth |
| :--- | :--- | :--- | :--- |
| `POST` | `/login` | Autenticarse con Email/Teléfono y Contraseña. | No |
| `POST` | `/register` | Registrar una nueva cuenta de usuario. | No |
| `POST` | `/forgot-password` | Solicitar un código de restablecimiento de contraseña. | No |
| `POST` | `/reset-password` | Establecer una nueva contraseña usando un código de restablecimiento. | No |
| `POST` | `/email-verification` | Verificar una dirección de correo electrónico usando un código. | No |
| `POST` | `/phone-verification` | Verificar un número de teléfono usando un código. | No |

::alert{type="info"}
Para ejemplos detallados de petición/respuesta, por favor consulta la interfaz Swagger si está habilitada en tu aplicación.
::
