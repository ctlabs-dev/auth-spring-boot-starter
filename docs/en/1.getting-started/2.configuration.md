Configure the starter in your `application.properties` or `application.yml`.

## Essential Configuration

These properties are required for the starter to function correctly.

```ini
# JWT Configuration
ctlabs.auth.jwt.secret-key=YOUR_SECURE_SECRET_KEY... # Must be a long, secure Base64 string
ctlabs.auth.jwt.expiration=86400000                  # Token expiration in milliseconds (default: 24 hours)

# Frontend URL (used in verification emails)
ctlabs.auth.frontend-url=http://localhost:3000
```

## Database Migration

The starter manages its own schema using Flyway.

```ini
# Enable automatic migration (default: true)
ctlabs.auth.db.migration-enabled=true
```

::callout{type="info"}
The starter uses a separate history table `ctlabs_auth_schema_history` to avoid conflicts with your application's migrations.
::

## Administrator Provisioning

Automatically create an admin user on startup.

```ini
ctlabs.auth.admin.enabled=true
ctlabs.auth.admin.email=admin@company.com
ctlabs.auth.admin.password=SecurePass123!
ctlabs.auth.admin.first-name=Super
ctlabs.auth.admin.last-name=Admin
```

## Notifications

Enable specific channels for user verification and password resets.

### Email (SMTP)

Requires standard Spring Boot Mail configuration.

```ini
ctlabs.auth.notifications.mail.enabled=true

# Spring Boot Mail Props
spring.mail.host=smtp.gmail.com
spring.mail.port=587
spring.mail.username=your@gmail.com
spring.mail.password=your_password
spring.mail.properties.mail.smtp.auth=true
spring.mail.properties.mail.smtp.starttls.enable=true
```

### SMS & WhatsApp (Twilio)

::code-group
  ```ini [SMS]
  # Twilio Credentials
  ctlabs.auth.twilio.account-sid=AC...
  ctlabs.auth.twilio.auth-token=...

  # SMS
  ctlabs.auth.notifications.sms.enabled=true
  ctlabs.auth.notifications.sms.phone-number=+1234567890
  ```

  ```ini [WhatsApp]
  # Twilio Credentials
  ctlabs.auth.twilio.account-sid=AC...
  ctlabs.auth.twilio.auth-token=...

  # WhatsApp
  ctlabs.auth.notifications.whatsapp.enabled=true
  ctlabs.auth.notifications.whatsapp.phone-number=whatsapp:+1234567890
  ctlabs.auth.twilio.whatsapp-content-sid=HX... # Required only if using WhatsApp Templates
  ```
::

## Customization

```ini
# Base path for auth endpoints (default: /api/auth)
ctlabs.auth.base-url=/v1/security

# Default role for new users (default: ROLE_CUSTOMER)
ctlabs.auth.default-role=ROLE_USER
```
