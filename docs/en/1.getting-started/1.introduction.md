Get started with the Auth Spring Boot Starter in under 5 minutes.

## Prerequisites

- Java 17 or later
- Spring Boot 3.x
- A database (PostgreSQL, MySQL, etc.)

## Installation

Add the dependency to your project.

::code-group
  ```xml [pom.xml]
  <dependency>
      <groupId>dev.ctlabs</groupId>
      <artifactId>auth-spring-boot-starter</artifactId>
      <version>0.0.1-SNAPSHOT</version>
  </dependency>
  ```

  ```groovy [build.gradle]
  implementation 'dev.ctlabs:auth-spring-boot-starter:0.0.1-SNAPSHOT'
  ```
::

## Basic Configuration

Add the following configuration to your `application.yml` or `application.properties`:

```yaml
ctlabs:
  auth:
    jwt:
      secret-key: ${JWT_SECRET:your-256-bit-secret} # Must be a long, secure Base64 string
      expiration: 86400000 # 24 hours in milliseconds
    frontend-url: ${FRONTEND_URL:http://localhost:3000}
```

## Run Your Application

The starter auto-configures all necessary components and creates the database schema automatically using Flyway.

You can now access the API documentation at `/api/auth` (or your configured base path).
