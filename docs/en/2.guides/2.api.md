The starter exposes the following endpoints under the configured base path (default: `/api/auth`).

| Method | Endpoint | Description | Auth |
| :--- | :--- | :--- | :--- |
| `POST` | `/login` | Authenticate with Email/Phone & Password. | No |
| `POST` | `/register` | Register a new user account. | No |
| `POST` | `/forgot-password` | Request a password reset code. | No |
| `POST` | `/reset-password` | Set a new password using a reset code. | No |
| `POST` | `/email-verification` | Verify an email address using a code. | No |
| `POST` | `/phone-verification` | Verify a phone number using a code. | No |

::alert{type="info"}
For detailed request/response examples, please refer to the Swagger UI if enabled in your application.
::
